Personal build is:
AMD FX-8350
Radeon R9 280X
3.5TB of storage on random hard drives and a boot SSD
Motherboard is ASUS M5A97 R2.0
10GB DDR3 1600MHz

All in all, I even kinda hate the fact that it still is so powerful and adequate - I really have no excuse to update it, especially considering I barely ever play games nowadays.
Built the computer in 2014 and to be honest I don't see myself upgrading it in a few more years. I am very much an enthusiast on computer building but that doesn't really make my wallet any thicker.

Recently got a small HTPC/Couch gaming computer for the living room with my girlfriend tho:
ASRock B450M Pro4
AMD A12-9800E
8GB DDR4 2133MHz
120GB M.2 SSD

CPU is pretty much just a placeholder - I "accidentally" won it at an online auction for 25€ and figured we should probably get a computer to put it into.
Later we also bought two 4TB hard drives and installed them, now we have a huge file server at home that we can access and use.

The APU's GPU isn't terribly amazing, but it actually even runs some games just fine, like Portal 2 on highest settings at 1080p. Everything it doesn't run we can just stream from my PC so it does give us pretty much all we need.

Now been wondering if we'll get the GTX 1650 and install it - would make the living room PC more powerful than mine which is kinda silly, but at least then we could game properly on the thing.
