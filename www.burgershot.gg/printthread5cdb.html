Not sure if anyone will find this useful but this parses for PAWN documentation in any pawn files and then create markdown files that is usable in GitHub Wiki. The tool was originally made by me to parse YSI for all the docs and create a kind ofÂ proper wiki.

How to use:
Basically just select the files you want to parse for pawn doc and then select an output folder and click that button and ta da.. You have a complete wiki.

Examples:
YSI by Y_Less - https://github.com/pawn-lang/YSI-Includes/wiki
Dynamic Doors by Crayder - https://github.com/Crayder/Dynamic-Doors#crayzie-dynamic-doors
