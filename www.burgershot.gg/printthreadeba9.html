RE: Dual monitors with San Andreas - LaszloR1 - 2021-03-18

Hi! It is very possible to use multiple monitors with SA-MP. Does GTA freeze in the not responding and then crashing sense or do you mean it just not moving while you're working on something on the other monitor? If the first it is a common issue with GTA. Some things that might fix it:
Running it in a compatibility mode. Like winxp, win7, etc. (Right click on gta_sa.exe, properties, compatibility)
You could also try to run it as an administrator.

If these don't work try looking online for more solutions.

RE: Dual monitors with San Andreas - LaszloR1 - 2021-03-18

I see, well. I remember opcodeexe making a mod where you can run multiple instances of SA-MP at the same time, but that only works on 0.3.7. I can't remember any other solutions to this sadly.

[Video: https://www.youtube.com/watch?v=_HnNub36mvY]

Edit: I looked into it a bit more and ThirteenAG's Widescreen fix contains an option to allow this.
https://github.com/ThirteenAG/WidescreenFixesPack/releases/tag/gtasa

After installing you need to navigate to the "scripts/" folder and open "GTASA.WidescreenFix.ini". You need to edit "AllowAltTabbingWithoutPausing = 0" to "AllowAltTabbingWithoutPausing = 1".
