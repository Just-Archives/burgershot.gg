The SAMP Servers API by Southclaws has a banner feature in it, which sadly no one uses yet. But can be implemented (or, merged) into the mod. which will allow servers use to their own thumbnails.

Creating a team to control the banners or logos is not a big thing and can be done by the wonderful community we have. Of course, there can be issues but this gives the mod a positive image.

The mod already is open-source and has no plans on monetization, hence can have people who can equally put their time into moderation as much as development.

Furthermore, we can have a limit that a server can only apply for a banner change once in a month, or unless they request from the team if they have good reputation.
