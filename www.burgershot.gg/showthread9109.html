Possibly the only good feature from Python is `for/else`. In pawn syntax:

PHP Code:
for (new i = 0; i != 10; ++i)
{
    if (Something(i))
    {
        break;
    }
}
else
{
    SomethingFailed();
}
// `break` jumps to here. 

The `else` block is only run if the loop exited without calling `break`, i.e. ended normally. I'm sure there's a way to fake or mimic it.
