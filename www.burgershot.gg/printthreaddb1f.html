Acerca de

Sublime Text 3 es un editor de texto altamente personalizable con muchas características geniales, que no puedes usar con el editor Pawno estándar o Notepad ++. Sé que ya hay un montón de tutoriales sobre esto, pero no explícitamente cómo instalar los paquetes muy bien, lo que puede ser confuso para los principios y las instrucciones sobre el uso del programa.
  • Puede personalizar completamente el editor a algo con lo que se siente cómodo.

  • Escribir código es más rápido debido a la terminación automática y una corrección automática inteligente.

  • Tu código se verá mucho más limpio con la sangría limpia que ofrece.

  • Una gran cantidad de atajos de teclado útiles (personalizables) para escribir código más fácilmente.

  • También funciona con muchos otros idiomas.

  • Sublime Text está disponible para Windows, OS X y Linux.

instalacion

Haga clic aquí -Descarga la última versión de Sublime Text.
Haga clic aquí para descargar el sintaxis de Sublime Text.


Sintaxis instalando

Descarga e instalación Sublime Text 3
Descarga la sintaxis de texto sublime.
Archivo de la sintaxis en Tu unidad de disco duro: \ Usuarios \ Tu nombre de usuario \ AppData \ Roaming \ Texto sublime 3 \ Paquetes gratuitos (SI NO SE VENDE LA CARPETA SUBLIME TEXT 3 ES ROAMING, ESCRIBAN LA DIRECCIÓN EN LA BARRA DE DIRECCIÓN  )
Abre Texto Sublime 3 (o reinicialo si estaba abierto)
Ve a Preferencias> Control de paquetes y elije Instalar Paquete y espera hasta que se abra una nueva ventana.
Busca "peón" y haz clic en Sintaxis de peón, luego espera en tu propia instancia (progreso en la esquina inferior izquierda)
Ahora puedes cambiar la sintaxis en un lugar en la esquina inferior derecha o en "Ver> Sintaxis> Peón"

Ahora puedes probar el mensaje SendClientMessage para ver si la función de auto-completado funciona. Si no funciona, los pasos erróneamente.

[font=Roboto, Arial, Helvetica, sans-serif]Instalancion del compilador
[/font]
Ve a "Herramientas> Construir sistema> Nuevo sistema de compilación"

Remplaza el código por este
Quote: {
"cmd": ["pawncc.exe", "-i includes", "$ file", "-; +"],
"path": "C: / path / to / pawno / folder"
}


[font=Roboto, Arial, Helvetica, sans-serif][font=Roboto, Arial, Helvetica, sans-serif]Cambia "" C: / path / to / pawno / folder "a donde tengas tu compilador (es la carpeta donde está el Pawno que tu sueles / quieres usar). Recuerda usar" / "y no" \ "porque si no el compilador dara un error.
[/font][/font]
Guarda el archivo en algún lugar de tu PC y cierra la ventana.

Reincia el programa.

Selecciona tu compilador: "Herramientas> Crear sistema> Nombre que elegiste"
[font=Roboto, Arial, Helvetica, sans-serif][font=Roboto, Arial, Helvetica, sans-serif]Ahora puedes usar Control + B para compilar el código (tecla predeterminada)[/font][/font]


Felicidades terminaste el tutorial.


Instrucciones extras.

¿Cómo cambiar el botón del compilador?
Ve a "Preferencias> Vinculaciones de teclas"
Busca la palabra "construir" usando Control + F
Reemplaza la tecla con la que quieras (Ej: f5) y copia la línea entera y pegala en el otro lado de la ventana entre los corchetes.
Guarda el archivo y reinicia Sublime Text 3.


¿Cómo cambiar el esquema de colores?
Ve a "Preferencias> Esquema de color"
Y elije el que mas te guste. ¿No te gusta ninguno? lee la siguiente pregunta.

¿Como obtengo nuevos esquemas de colores?
Ve a "Preferencias> Control de paquetes" y elije "Instalar paquete"
Puedes elegir entre esos.

Tabulación
Ve a "Ver> Sangría"
Recomiendo Ancho de Pestaña: 4 (es el mas usado ...)
La tabulación puede ser usada con "Ctrl + [" y "Ctrl +]" o con el botón TAB.
