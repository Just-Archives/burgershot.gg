RE: open.mp client download? - JustMichael - 2019-11-29

To add onto what Banditul has already said, open.mp will first release with a server only (drop-in replacement for sa-mp) package, that will help to get server owners over to us with ease. Only after this stage, will we be working on a Client that will eventually allow us to move away from the legacy sa-mp API and instead towards our own API that will leverage newer technologies.

So to be it short, yes there will be a Client, but it will be coming in the 2nd release of open.mp.
