This can help:

PHP Code:
IsVehicleOccupied(vehicleid)
{
    for(new i = GetPlayerPoolSize(); i != -1; i--) 
        if(IsPlayerInVehicle(i, vehicleid) && GetPlayerVehicleSeat(i) == 0)
            return 1;

    return 0;
} 

can use foreach too.
