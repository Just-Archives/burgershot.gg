RE: Nombre menor de 3 caracteres en SA:MP - Leonardo - 2021-05-29

Para tener menos de 3 caracteres, puedes parchar el ejecutable del servidor con algún editor hexadecimal.

Para windows, sería buscar lo siguiente:
Code:
8A 44 24 1F 3C 03 (npcs)
8A 44 24 1E 3C 03 (players)
y reemplazar el 03 por el número mínimo de caracteres.

Para linux, sería buscar lo siguiente:
Code:
0F B6 85 82 FC FF FF 3C 02 (npcs)
0F B6 85 92 ED FF FF 3C 02 (players)
y reemplazar el 02 por el número mínimo de caracteres.

En windows se calcula de la siguiente manera:
Code:
if(length < 3)
{
// error length
}

Mientras que en linux se calcula de la siguiente manera:
Code:
if(length <= 2)
{
// error length
}

Es algo a tener en cuenta. También se puede parchar desde un plugin o desde un include usando amx_assembly.

No es posible tener más de 24 caracteres.
