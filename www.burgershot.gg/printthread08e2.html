Yes, but
  • Make it toggle-able by the client;
  • Make it with servers running with 1000/1000 in mind (i.e. With performance in mind)
  • Make it as customizable by the server as possible. An example of this would be a SendVoiceToPlayer(playerid, targetid) which would enable all voice sent by playerid to be sent to targetid, provided they both enabled the voice chat. This way, the voice could be used for police radios, faction chats, proximity range etc.
